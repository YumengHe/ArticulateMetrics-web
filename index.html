🔧 Type of Joints 🔧

Articulated objects move through a set of canonical joint (link) types, each of which constrains how one part moves relative to another. The joint types below follow the constraint library of Project Chrono. The Constrained DOF column counts how many of the 6 relative degrees of freedom the joint removes, so a Fix joint constrains all 6 and a Free joint constrains none.

Joint Type Constrained DOF Description Example
Fix 6 Fixes both position and rotation; the two parts are rigidly attached.

Bushing 6 | 3 Linear compliance with an optional spherical joint.
Rubber suspension bushing
Revolute 5 Allows rotation along the Z axis only.
Door hinge
Prismatic 5 Allows translation along the Z axis only.

Universal 4 Universal joint that allows rotation about the X and Y axes.
Universal (Cardan) joint
Revolute + Prismatic 4 Permits translation along the X axis and rotation about the Z axis.

Oldham 4 Oldham joint; does not fix the axes' position.

Cylindrical 4 Keeps the Z axes collinear (rotation and translation along Z).

📚 Related Work 📚

Prior work on articulated 3D objects spans reconstruction, generation, and real-to-sim modeling. Each line below lists a representative paper and the quantitative metrics it uses to evaluate articulation quality.

Paper Venue Metrics Used
Articraft arXiv 2026 User study (prompt alignment & overall quality); downstream Particulate metrics: gIoU, mIoU, part-wise Chamfer (PC), whole-object Chamfer (OC)
Particulate CVPR 2026 gIoU; mIoU; part-wise Chamfer distance (PC); whole-object Chamfer distance (OC)
MeshArt CVPR 2025 ID / AID; COV; MMD; 1-NNA; FID; KID
Articulate AnyMesh arXiv 2025 Joint axis angular error; joint axis position error; manipulation success rate
SINGAPO ICLR 2025 Chamfer Distance (dCD); generalized-IoU; centroid distance; AOR; ID / AID; graph-topology accuracy
Real2Code ICLR 2025 Chamfer Distance (whole / part); joint axis rotation error; joint position error; joint type accuracy
Articulate-Anything arXiv 2024 Link / joint prediction success rate; joint type, axis, and origin error; VLM realism rating
URDFormer RSS 2024 Mesh accuracy; parent accuracy; spatial error; precision / recall / F1; manipulation success rate
CAGE CVPR 2024 MMD / COV under ID and AID (Abstract Instantiation Distance); AOR (Average Overlapping Ratio); 1-NNA
NAP NeurIPS 2023 Instantiation Distance (ID); MMD; COV; 1-NNA
PARIS ICCV 2023 Chamfer Distance (whole / static / movable); axis angular error; axis position error; geodesic error (revolute); translation error (prismatic)
CARTO CVPR 2023 Chamfer Distance; mAP @ 3D IoU25 / IoU50; mAP @ pose thresholds; joint type accuracy; joint state error
Reart CVPR 2023 Chamfer Distance / reconstruction error; Rand Index; Tree Edit Distance; scene-flow EPE (Acc5 / Acc10 / angle error); retargeting error

Watch It Move CVPR 2022 LPIPS; SSIM; MPJPE (mean per-joint position error)

SAPIEN / PartNet-Mobility CVPR 2020 mAP for movable-part segmentation; motion-axis alignment; joint pivot / position error; joint type classification
